# BentoLog - Engineering Journal

A sleek, bento-style engineering journal for developers and architects.

## Features
- **Markdown Rendering:** Powered by `marked.js` with metadata (frontmatter) support.
- **Syntax Highlighting:** Integrated with `highlight.js` for code blocks.
- **Search:** Instant search across all posts.
- **Theming:** Light/Dark mode support.
- **Responsive Design:** Mobile-first approach with interactive sidebar.
- **Categories:** Dynamic category filtering based on post tags.

## Project Structure
- `index.html`: Main entry point and layout.
- `style.css`: Custom styles (modern, bento-grid inspired).
- `script.js`: Core logic for fetching posts, parsing Markdown, and search.
- `posts/`: Directory containing `.md` files for blog content.

## Getting Started
Simply open `index.html` in your browser. For local development with external file fetching, it's recommended to use a local development server (e.g., `Live Server` in VS Code or `python -m http.server`).
